# 🚀 CalendarBot – AI-Powered Calendar Assistant

CalendarBot is a cloud-native AI assistant that allows users to interact with their Google Calendar using natural language. Instead of navigating manually, users can simply ask questions like:

> “What’s my schedule tomorrow?”

and receive instant responses.

---

## ✨ Features

- 💬 Natural language-based calendar queries  
- 📅 Google Calendar API integration  
- ☁️ Cloud deployment using WSO2 Choreo  
- 🚦 API rate limiting for stability  
- ⚡ Real-time schedule retrieval  

---

## 🧠 How It Works

1. User sends a query via chatbot  
2. Backend processes the input  
3. System interprets intent  
4. Google Calendar API is called  
5. Response is returned to the user

---

## ⚙️ Tech Stack

- Backend: Node.js, TypeScript  
- APIs: Google Calendar API  
- Cloud: WSO2 Choreo  
- Others: REST APIs, Rate Limiting, Tuari

---

## 🚀 Deployment

The backend is deployed using WSO2 Choreo, providing:

- Managed cloud deployment  
- Scalable backend services  
- API exposure and control
